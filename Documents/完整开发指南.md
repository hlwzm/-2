# 指尖江湖2 —— 完整开发指南

## 一、项目架构总览

### 技术栈
- 客户端：Unity 2022.3.62f2c1 (C#)
- 服务端：.NET 9 (C#) 微服务架构
- 通信：TCP (Binary协议, Magic 0x9A7B)
- 版本控制：Git (main分支)

### 项目目录
指尖江湖2/
├── Client/                    # Unity客户端
├── Server/                    # 微服务（15个项目）
│   ├── Jx3.MockServer/        # 整合Mock服务
│   ├── Jx3.Battle/            # 战斗结算
│   ├── Jx3.Chat/              # 聊天
│   ├── Jx3.Dungeon/           # 副本
│   ├── Jx3.Trade/             # 交易行
│   ├── Jx3.Shop/              # 商城
│   ├── Jx3.Social/            # 社交(好友/帮会)
│   ├── Jx3.PVP/               # PVP竞技
│   ├── Jx3.Quest/             # 任务
│   └── Jx3.Common/            # 公共库(协议定义)
├── Documents/                 # 设计文档
├── BuildLogs/                 # 编译日志
└── Tools/                     # 辅助脚本

## 二、启动方式

### 启动MockServer
```bash
cd Server
dotnet run --project Jx3.MockServer/Jx3.MockServer.csproj -c Release
```

### 启动Unity客户端
- 打开 Unity Hub → 选择 Client/ 目录
- 或直接：D:\LearnSoftWare\Unity2022\2022.3.62f2c1\Editor\Unity.exe -projectPath D:\CodexWorkSpace\MyGame\指尖江湖2\Client

### 编译验证
```bash
# Unity命令行编译
D:\LearnSoftWare\Unity2022\2022.3.62f2c1\Editor\Unity.exe -quit -batchmode -projectPath D:\CodexWorkSpace\MyGame\指尖江湖2\Client

# 服务端编译
cd Server && dotnet build Jx3.sln -c Release
```

## 五、编码规范速查
- UI: 全程序化生成，继承BasePanel，调用base.Awake()
- 主题: 暗黑紫 #0a0a14/#12121e/#b088ff
- Font: Resources.GetBuiltinResource<Font>("LegacyRuntime.ttf")
- Image: new GameObject(name, typeof(RectTransform), typeof(Image))
- Manager: 单例模式，Instance属性
- 命名空间: Jx3.Core / Jx3.UI.Panels / Jx3.Core.Battle 等
